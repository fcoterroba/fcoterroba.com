**WHAT ARE PWAs?**

A **progressive web application** (PWA for its acronym in English) is a type of application software that is delivered through the web, created using common web technologies such as HTML, CSS and JavaScript. It's intended to work on any platform that uses a standards-compatible browser. The functionality includes working offline, push notifications and access to device hardware, which allows creating user experiences similar to native applications on mobile and desktop devices. Since a progressive web application is a type of web page or website known as a web application, there's no requirement for developers or users to install web applications through digital distribution systems such as Apple App Store or Google Play.

The graphical interface would be the same as the one available on the web with some additional advantages such as push notifications, being able to visit the application without internet access, access to device hardware, etc.
