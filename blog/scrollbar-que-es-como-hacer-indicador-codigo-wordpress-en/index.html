Although it doesn‚Äôt specifically refer to the following, a scrollbar is the scroll bar located normally on the right of all web pages and applications that shows when you‚Äôll reach the end of that web page.

Last and most important, the script, which calculates and displays, according to the screen width everything put inside a simple function.

window.onscroll = function(){ myFunction(); };

function myFunction(){
  var winscroll = document.body.scrollTop || document.documentElement.scrollTop;
  console.log(winscroll);
  
  var height = document.documentElement.scrollHeight - document.documentElement.clientHeight;
  console.log(height);
  
  var scrolled = (winscroll/height) * 100;
  document.getElementById("scrollIndicator").style.width = scrolled + "%";
}
